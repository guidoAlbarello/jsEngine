# sistemasGraficos
```
Controles:
	- Z : Switchea entre modo desarrollador y modo simulacion. En modo desarrollador muestra una grilla, los ejes en cada objeto, y los modelos de desarrollo de los objetos (por ejemplo, las camaras tienen un frustrum y un cubo para representar su posicion). 
	- X : Esconde los ejes en modo desarrollador.
	- C : Se cambia entre las camaras de la escena. (Solo cambiar de camara fuera del modo desarrollador, ya que sino al cambiar entre las camaras se vera el cubo del modelo).
	- En las camaras orbitales se gira la camara presionando el boton derecho o izquierdo del mouse y moviendolo. Tambien se puede hacer zoom scrolleando la rueda del mouse. 
	- Para la camara primera persona se mueve con las teclas W,A,S,D y Q y E para elevar o descender. La camara se mueve moviendo el mouse.
```

```
Camaras:
	- Camara orbital global de la escena.
	- Camara primera persona.
	- Camara orbital centrada en la montaña rusa.
	- Camara desde el punto de vista del carrito.
	- Camara orbital centrada en el carrito (extra para poder ver el carrito mejor).
	- Camara orbital centrada en el carousel.
```
