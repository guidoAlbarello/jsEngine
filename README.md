# sistemasGraficos
```
Controles:
	- Z : Heavy attack
	- X : Light attack
        - E : Dash attack 
        - W: Up/Jump
        - S: Down
        - D: Right
        - A: Left
        - Q: Shoot
Combos:
        - Uppercut: Heavy_Attack + Up
	- Sky slam: Heavy_Attack + Down (in air)
	- Attack1: light_attack
	- Attack2: light_attack + light_attack (after 1 second) 
	- Dash attack: Dash_attack + direction
	
	This combos can be combined in almost any way.
